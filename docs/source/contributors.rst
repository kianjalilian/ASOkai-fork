Contributors
============

ASOkai is built by a small team of researchers and engineers. The following
people are responsible for the ongoing development and maintenance of the
project:

- Arash Ayat -- Author
- Nathalie Gocht -- Maintainer
- Aleksandra Khatova -- Maintainer
- Kian Jalilian -- Documentation

**Community contributions**

We welcome community contributions of any size, from documentation fixes to new
features and bug reports. If you would like to get involved:

- Review the :doc:`Getting Started <getting-started>` guide to understand the
  project structure.
- Open an issue or pull request on GitHub at
  https://github.com/Schlieplab/genome_utils.
- See the ``CONTRIBUTING`` guidelines in the repository (or open a discussion)
  if you have questions about workflows, coding style, or release planning.

**Acknowledgements**

Thanks to everyone who has reported issues, suggested improvements, or tested
early releases. Your feedback helps shape the direction of GenomeUtils.
